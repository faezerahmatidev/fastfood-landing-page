# 🍔 Foodeli – Fast Food Landing Page

A modern, clean, and fully responsive fast‑food landing page inspired by McDonald's design language.  
Built with HTML, CSS (Flexbox), and vanilla JavaScript — optimized for performance, clarity, and scalability.

---

## 🚀 Live Demo

https://faezerahmatidev.github.io/foodeli-landing-page/

---

## 📌 Project Overview

Foodeli is a visually appealing fast‑food landing page featuring a clean UI, promotional banners, product cards, digital app promotions, and brand‑focused sections.  
The layout is fully responsive and designed to work seamlessly on desktop, tablet, and mobile screens.

This project is ideal for:

- Practicing front‑end layout techniques
- Practicing UI/UX structure
- Building modern landing pages
- Portfolio showcases

---

## 🧩 Features

- Fully responsive layout (Desktop → Tablet → Mobile)
- Smooth UI layout using CSS Flexbox
- Clean and modern fast‑food branding
- Reusable product card components
- Hero promotion section
- App download call‑to‑action (Google Play / App Store)
- Lightweight and optimized performance
- Well‑structured, scalable folder organization

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
  - Flexbox
  - Responsive Layouts
  - Grid Layout
- **JavaScript (Vanilla)**
- **Google Fonts**
- **Modern UI/UX design principles**
